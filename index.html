NewOtangle Web Graph

Ultimate Network Intelligence & Control Platform

Otangle Web Graph is a Network Security & Compliance Platform. It enables businesses to accurately and in real-time monitor users and user actions in their networks. It provides network processes compliance checking and enforcement. By recognizing non-compliant actions Otangle Web Graph automatically initiates a response, ranging from alerting the security team to targeted mitigation.

Technology 

Otangle Web Graph Platform:

Platform is based on core modules

Otangle Validation Authority and Otangle Software Agents

Otangle web graph provides real-time compliance checking and network protection through the close integration of innovative patented and patent pending technologies. Its key components are a high-throughput validation Authority (VA) and ultra-light weight, easy to install distributed software agents (SA). The system allows to monitor users and processes at wire speed at the network layer. It is able to provide network as well as application security.

Otangle Validation Authority (VA)

  1. Receives observations from distributed software agents installed on endpoint devices.
  2. Builds an event trace in real time for compliance checking.
  3. Effectively applies a policy and rules sets to the event trace.
  4. Acts like a referee in a sports game processing observations which are provided by the SAs to monitor and enforce compliance.
  5. Designed for high-throughput - does not introduce latency; computing resources are focused on resolving anomalies and enforce compliance.
  6. Capable to verify compliance from the network to the business process level

Otangle Software Agents (VA)

  1. SAs mine ingoing and outgoing data packets at the lowest possible communication layer and transmit observations to the VA.
  2. SAs are provided for virtual machines in the cloud, mobile devices as well as workstations and laptops.

These features enable the efficient and automated detection and resolution of anomalies in distributed business processes.

Otangle Access Manager

Based on advanced patented and patents pending technology Otangle Access Manager (AM) is designed to protect significant network assets without introducing latency. Incoming and outgoing packets are checked at wire speed both for compliance checking and access control purposes. In combination with VA suspicious activities are attributed to specific users and process flows in real time. This actionable information enables a targeted response including silent monitoring, stopping transactions, disabling a user’s access to specific resources, or on-the-fly re-prioritizing user access to network resources according to specific user profile features. AM can be configured to protects assets both at the network and application level.
